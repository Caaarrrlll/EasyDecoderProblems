	```
	g++ <filename>.cpp -o <filename>
	```

	```
	./<filename>

	```
	Ceil And Max: <double toCeilAndFloor>

	```

	```
	The Beta Test Intitiation: <double minAndroidVer> <double currAndroidVer>    
	```  
	```
